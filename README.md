# Birahe University Landing Page

This project represents my first experience in web development and is a custom-designed landing and home page created for the **Birahe student competition** at **Kashan University**.  
The primary goal of this project was to design a **clean, modern, and responsive** user interface suitable for academic presentations while practicing Front-End development skills.

## Key Features
- Fully responsive design compatible with mobile, tablet, and desktop devices  
- Simple, clear, and professional user interface  
- Efficient utilization of HTML, CSS, and JavaScript  
- Organized and optimized file structure adhering to standard web development practices
